# 蟹塘水草监测系统

## 系统简介

蟹塘水草监测系统是一个专为水产养殖户设计的智能监测平台，通过内置的预测模型分析温度、溶解氧等环境数据，预测水草覆盖度并提供及时的收割建议，帮助养殖户优化蟹塘管理，提高养殖效率。

## 核心功能

### 1. 实时数据监测
- 水温、溶解氧、水草覆盖度等关键指标实时展示
- 迷你图表直观显示最近7天数据趋势
- 同比数据对比，了解变化情况

### 2. 智能预测分析
- 基于多项式回归模型预测未来7天水草覆盖度
- 提供置信区间，展示预测的可靠性
- 当预测覆盖度超过阈值时自动发出预警

### 3. 模型管理
- 支持查看当前模型状态和精度
- 可调整训练参数（训练周期、学习率）重新训练模型
- 支持通过权重调整优化模型性能

### 4. 数据导入导出
- 支持CSV格式数据导入，快速更新监测数据
- 可导出历史数据，方便进行离线分析
- 提供示例数据文件 `sample_data.csv`

### 5. 智能建议
- 根据预测结果自动生成收割建议
- 提供分区收割、施肥等优化建议
- 建议内容包括具体时间、原因和详细操作指南

## 使用指南

### 快速开始

1. 打开系统主页面 `index.html`
2. 系统会自动加载模拟数据并初始化预测模型
3. 在仪表盘页面查看当前监测数据和预测结果
4. 点击"模型管理"查看模型状态和精度

### 数据导入

1. 点击左侧导航栏的"模型管理"
2. 选择"手动上传"更新方式
3. 点击"浏览文件"选择CSV格式的数据文件
4. 确保数据包含以下字段：`timestamp`, `temperature`, `dissolvedOxygen`, `coverage`
5. 点击"确认更新"导入数据并更新模型

### 模型训练

1. 在仪表盘页面点击"更新模型"按钮
2. 调整训练参数（训练周期、学习率）
3. 选择训练数据范围
4. 点击"开始训练"，系统会显示训练进度和损失曲线
5. 训练完成后，系统会自动更新预测结果

### 查看预测结果

1. 在仪表盘页面查看水草覆盖度预测图表
2. 图表中蓝色虚线表示预测覆盖度，阴影区域表示置信区间
3. 红色虚线表示建议收割阈值（85%）
4. 当预测覆盖度超过阈值时，系统会在顶部显示预警信息

## 技术实现

### 预测模型

系统采用二阶多项式回归模型，考虑了以下因素对水草生长的影响：
- 水温（主要影响因素）
- 溶解氧含量（与温度负相关）
- 历史覆盖度（反映生长趋势）

模型通过梯度下降算法进行训练，支持动态调整各因素的权重。

### 数据处理

系统会对输入数据进行以下处理：
1. 特征扩展：将原始特征扩展为多项式特征
2. 数据标准化：确保各特征在合理范围内
3. 异常值检测：识别并处理异常数据点

### 智能建议生成

系统基于以下规则生成智能建议：
1. 当预测覆盖度超过85%时，建议安排收割
2. 当预测覆盖度超过90%时，发出警告并建议紧急处理
3. 当预测覆盖度低于50%时，建议施肥促进生长
4. 正常情况下，建议定期监测维护

## 示例数据

```csv
...
```
